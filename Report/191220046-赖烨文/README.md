#### 3.问题

##### 使用git的好处：

建立版本库，方便随时回退与修改

给项目创建分支，方便多人项目开发

##### 使用远程仓库的好处：

可以高效在线协同开发

建立数据云备份，防止数据丢失损坏

##### 在开发中使用分支的好处：

方便同一项目不同功能开发的同步进行

在实际开发中，我们团队可以为每个功能实现创建一个分支，大家只需在小组范围内处理冲突问题，有效加速开发效率
