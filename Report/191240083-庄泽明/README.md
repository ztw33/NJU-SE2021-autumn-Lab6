## 实验结论

* 使用git的好处？

  - 对项目的管理便捷，体现在分支和log上
  - pull和push方便

* 使用远程仓库github的好处？

  - 能够较快的推送和拉取需要存放的文件
  - 对多人合作的项目处理方便
  - 可以学习别人公开的项目学习

* 在开发中使用分支的好处？你在开发中有哪些体会和经验？

  - 不用对一个文件的不同版本分别进行保存
  - 遇到bug可以查看log进行回溯
  - 可以减少不同阶段的代码修改之间的影响
