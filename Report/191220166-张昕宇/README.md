## 问题回答

1. 使用git的好处？

   使用git可以很方便地进行版本控制，记录每次改动，做改动时不用担心影响原有的稳定分支/版本。

2. 使用远程仓库的好处？

   使用远程仓库可以备份自己的代码，同时可以很方便地与他人协作开发。

3. 在开发中使用分支的好处？

   可以在稳定的版本上进行分支开发，方便调试，同时也不用担心开发调试时其他分支功能受到影响。
