• 在报告中回答以下问题： 

• 使用 git 的好处？ 

便于管理代码，方便进行版本后退，可以建立分支对代码进行修改

• 使用远程仓库 (如 github/gitee 等) 的好处？ 

方便多人协同开发，可以备份代码和共享

• 在开发中使用分支的好处？你在实际开发中有哪些体会和经验？ 

方便多人共同开发，不同的人在不同分支上开发，避免冲突
